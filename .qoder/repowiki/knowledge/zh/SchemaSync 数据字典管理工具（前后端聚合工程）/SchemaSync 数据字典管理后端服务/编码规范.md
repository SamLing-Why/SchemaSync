- 每个数据库方言以独立类实现 `DatabaseAdapter` 接口并通过工厂 `DatabaseAdapterFactory` 按类型名选择，新增方言只需添加实现类并在工厂扫描范围内注册。
- 领域对象集中在 `model/dict` 与 `model/diff` 包下，作为跨层共享的纯数据结构，不包含业务方法。
- 所有对外 API 由 `@RestController` 暴露，控制器仅做参数校验与调用 service，无直接 SQL/JDBC 访问。
- 配置文件按环境拆分 `application-{dev,prod}.yml`，通过 Maven profile 的 `spring.profiles.active` 属性激活。
- 日志统一输出到 `logs/schemasync.log` 并按天滚动压缩（max-history: 30），应用启动时在 main 中打印版本信息。
