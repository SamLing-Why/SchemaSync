单页应用，入口 `src/main.js` 通过 `createApp(App)` 挂载根组件，并全局注册 ElementPlus、vue-router、vue-i18n 以及遍历 `@element-plus/icons-vue` 批量注册所有图标。
- 路由层 `src/router/index.js` 使用 `createWebHistory` 扁平声明 `/config`、`/export`、`/diff`、`/generate`、`/settable` 五个业务路由，根路径重定向至 `/config`。
- 视图层 `src/views/` 下每个 `.vue` 对应一个功能页面；通用 UI 片段抽取为 `src/components/DiffDetail.vue`、`LanguageSwitcher.vue`。
- 网络层 `src/api/request.js` 基于 axios 创建实例，统一 baseURL 为 `/api`、超时 30s，并通过响应拦截器将后端返回的 data 透传、错误统一用 ElMessage 提示；`vite.config.js` 中 dev server 在端口 3000 启动并将 `/api` 代理到后端 `http://localhost:8999`。
- 国际化 `src/locales/index.js` 以 vue-i18n Composition API 模式初始化，默认语言从 localStorage 读取或回退到 `zh-CN`，消息文件按 `zh-CN.js` / `en-US.js` 拆分。
依赖方向：views → components → api/request；main.js 作为唯一装配点，router/i18n 通过 app.use 注入。
