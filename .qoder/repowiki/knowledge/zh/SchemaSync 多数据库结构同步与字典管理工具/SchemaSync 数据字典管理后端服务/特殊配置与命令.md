Maven 构建时通过 `frontend-maven-plugin` 在父目录 `../schemasync-frontend` 安装 Node.js v18.19.0 + npm 10.2.3 并执行 `npm install && npm run build`，再将 `dist` 产物复制到 `target/classes/static`；打包使用 `spring-boot-maven-plugin` 且 layout=ZIP 以支持 `-Dloader.path` 外部化扩展。运行命令示例：`mvn clean package -P prod` 或 `java -jar target/schemasync-backend-1.0.6.jar --spring.profiles.active=prod`。
