根目录作为 Maven + Vite 双子工程编排层：schemasync-backend 为 Spring Boot REST API，通过 adapter 层的 DatabaseAdapter 接口统一对接 MySQL/Oracle/GaussDB/GoldenDB/OceanBase/TDSQL 等数据库；schemasync-frontend 使用 Vite 开发时通过 proxy 转发到后端，生产构建后由后端静态资源映射或 Nginx 反向代理部署。前后端契约集中在 controller 包暴露的 REST 接口上，前端通过 src/api/request.js 发起 HTTP 调用，共享的领域模型以 JSON 在 model/dict 与 model/diff 包中定义并由双方复用语义。顶层 deploy/ 提供 application.yml、start.sh/bat 与打包产物 schemasync_1.0.3.jar，构成统一的发布单元。
