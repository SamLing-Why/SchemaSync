- HTTP 请求统一通过 `src/api/request.js` 导出的 axios 实例发起，禁止在组件内直接 new axios()。
- 路由采用扁平式声明：在 `router/index.js` 中以对象数组集中定义 path/name/component，无嵌套路由。
- 国际化 key 按语言拆分为独立模块（`zh-CN.js`、`en-US.js`），由 `locales/index.js` 聚合并以 `locale` 字段驱动切换。
- Element Plus 图标通过遍历 `@element-plus/icons-vue` 的导出键名批量 `app.component(key, component)` 全局注册，而非按需 import。
